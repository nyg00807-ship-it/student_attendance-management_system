# 학생 출석 관리 시스템

## 프로젝트 개요
외부 서버에서 제공하는 학생 출석 데이터를 받아 화면에 표시한 예제입니다.

## 파일 구성
* server-data-js : 외부 학생 관리 서버에서 제공하는 학생 데이터
* index.html : 메인HTML파일
* style.css : 스타일시트
* script.js : 내부 처리 로직

## 실행 방법
1. 모든 파일을 같은 폴더에 저장합니다.
2. index.html 파일을 웹 브라우저로 엽니다
3. 필터 버튼을 클릭하여 상태별로 학생을 조회합니다.

## 주요 기능
* 외부 학생 관리 서버 데이터를 받아서 학생 카드 목록으로 표시
* 각 학생의ID, 이름, 반, 출석 상태 표시
* 상태별 필터 버튼으로 전체 / 출석 / 결석 / 지각 학생만 조회
* 상태에 따른 색상 표시(전체: 파랑 | 출석: 초록 | 결석: 빨강 | 지각: 노랑)
